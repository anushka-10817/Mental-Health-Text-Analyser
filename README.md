# Mental-Health-Text-Analyser
## Objective
The AI-Based Mental Health Text Analyzer is a machine learning project that analyzes user-input text and predicts the emotional state behind it. The goal of this project is to provide basic mental health insights by detecting emotions such as happiness, sadness, anger, and stress from textual data.

This project demonstrates the application of Natural Language Processing (NLP) and Machine Learning in solving real-world problems related to mental well-being.
## Features
* Detects emotions from the text input
* Classifies the text input into categories like Happy, Sad, Angry, Stressed
* Uses Machine Learning for prediction
* Simple and user-friendly interface
* Can be extended into a web app or chatbot
## Tools Used 
* Python 
* Scikit-learn
* TfidfVectorizer 
* Logistic Regression
* TextBlob
* Modules

 ## Limitations 
* Uses a small dataset, so accuracy may be limited
* Cannot handle mixed emotions effectively
* May misclassify complex sentences
* Limited understanding of context or slang
* Always assigns an emotion, even for neutral text
